# GlycoNutri 项目开发规范

## 开发流程
需求 → 设计 → 开发 → 测试 → 部署

---

## 当前版本: v1.1 ✅

已完成功能:
- CGM 数据分析 (TIR, GV)
- GI/GL 数据库 (50+ 食物)
- 餐后血糖分析 + PK/PD 指标
- 餐食营养分析
- 运动/睡眠血糖分析
- 药物血糖分析 (20+ 口服药 + 6 种胰岛素)
- 趋势分析 (日/周/月/时段)
- 图表可视化 (曲线图、饼图)
- 数据导出 (CSV)

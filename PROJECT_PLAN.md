## 技术方案

### 语音输入
- 方案: OpenAI Whisper (本地)
- 模型: base (英文) / large-v3 (多语言)
- 用途: 语音记录餐食、运动

### 图片识别
- 方案: 阿里云图像识别 + 本地食物库
- 功能: 食物分类、营养估算
